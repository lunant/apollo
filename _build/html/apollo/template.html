apollo.template — Templating¶

class apollo.template.Result(__value__={}, **items)¶

Bases: werkzeug.datastructures.ImmutableDict

The result dictionary for templating, which is immutable.

It combines key-value pairs for templating and a single value (__value__) for serialization formats like JSON, YAML.

>>> result = Result(123, a=1, b=2, c=3)
>>> result
apollo.template.Result(123, a=1, b=2, c=3)
>>> result.value
123

The argument __value__ of the constructor can be omitted. In these case, value returns just itself.

>>> result = Result(a=1, b=2)
>>> result
apollo.template.Result({'a': 1, 'b': 2})
>>> result.value
apollo.template.Result({'a': 1, 'b': 2})
Parameters:
  • __value__ – A single value for serialization formats, or a mapping object. Default is {}.
  • **items – Dictionary items.
