apollo.handler — Request handler¶

class apollo.handler.Context¶

The namedtuple() type that represents a request context.

It has three tuple fields in order: request, application and map_adapter:

assert isinstance(context, Context)
request, application, map_adapter = context
Parameters:
  • request (Request, werkzeug.BaseRequest) – a request object
  • application (Application) – an application instance
  • map_adapter (apollo.routing.MapAdapter) – a map adapter instance
request¶

The instance of Request which is a subclass of werkzeug.BaseRequest. (The first field of the tuple.)

application¶

The Application instance. (The second field of the tuple.)

map_adapter¶

The apollo.routing.MapAdapter instance. (The third field of the tuple.)

app¶

Alias of application.

args¶

Alias of request.args.

form¶

Alias of request.form.

values¶

Combined ImmutableMultiDict for args and form. Alias of request.values.
